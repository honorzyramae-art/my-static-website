CSS Selectors: Tag, Class, and ID

Tag selectors target all elements of a type.
Class selectors (e.g. .highlight) are reusable across different elements.
ID selectors (e.g. #site-title) are unique for one element per page.

Pro Tip

Use semantic tags for accessibility and SEO, ID selectors for uniqueness, and class selectors for flexibility and reuse.
